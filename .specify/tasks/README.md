# Tasks

This directory holds individual task files generated by the speckit
`/tasks` agent. Each file represents one implementation unit.

## Usage

Tasks are created by running `/speckit.tasks` in Copilot Chat or by
the speckit tasks agent. Each task file contains:

- Task description
- Dependencies on other tasks
- Acceptance criteria
- Files to modify

## Naming Convention

```
001-<short-description>.md
002-<short-description>.md
```

Tasks are numbered for dependency ordering.
